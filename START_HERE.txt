## ❓ Common Questions

### Q: Do I need to install anything?
**A:** No! Just open index.html in your browser.

### Q: Will my data be lost?
**A:** No! Data saves automatically and stays on your device.

### Q: Can I use this offline?
**A:** Yes! Complete offline support with local storage.

### Q: How do I backup data?
**A:** Press Ctrl+E to export data as JSON file.

### Q: Can I use it on my phone?
**A:** Yes! Fully responsive mobile design.

### Q: Do I need Firebase?
**A:** No! Optional for cloud sync. Works great offline.

## 🆘 Troubleshooting

### App not opening?
- Try different browser (Chrome recommended)
- Check file is named exactly `index.html`
- Clear browser cache (Ctrl+Shift+Delete)

### Data not saving?
- Check browser storage is enabled
- Try incognito/private mode
- Check file permissions

### Charts not showing?
- Refresh page (F5)
- Add some data first
- Check browser console (F12)

## 🎨 Customization

The app is built to be easy to customize:

### Change Colors
- Open: `css/style.css`
- Look for: `--primary-color: #2ecc71;`
- Change: Any hex color

### Add More Vaccine Types
- Open: `index.html`
- Find: `<select id="vaccine-type">`
- Add: `<option value="Your Vaccine">Your Vaccine</option>`

### Modify Fields
- Edit forms in: `index.html`
- Update modules in: `js/` files
